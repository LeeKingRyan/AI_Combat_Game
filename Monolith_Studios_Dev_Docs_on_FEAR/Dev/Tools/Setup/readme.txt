TYPEMAP
This directory contains tools to get "p4 typemap" setup and instructions on
how to apply this typemap to the appropriate files. Directions:
- run "p4 typemap"
- paste in contents of typemap.txt (this fixes it for new files)
- run P4-SetTypeMap.bat < typemap.txt (in //Tools/Scripts, requires Perl)
- P4: submit the changelist
