SoundNonPoint
SoundNonPoint is used to specify a sound to be played from a non-point source. Examples of it's use are playing water sounds flowing through the length of a pipe.

Message Description Example
VISIBLE <bool> Toggles whether the object is visible. msg <ObjectName> (VISIBLE 1)<BR>msg <ObjectName> (VISIBLE 0)
SOLID <bool> Toggles whether the object is solid. msg <ObjectName> (SOLID 1)<BR>msg <ObjectName> (SOLID 0)
HIDDEN <bool> Toggles whether the object is visible, solid, and rayhit. msg <ObjectName> (HIDDEN 1)<BR>msg <ObjectName> (HIDDEN 0)
REMOVE Removes the object from the game permanently. msg <ObjectName> REMOVE
CASTSHADOW <bool> Toggles whether the object casts a shadow. msg <ObjectName> (CASTSHADOW 1)<BR>msg <ObjectName> (CASTSHADOW 0)
SETPOS <X> <Y> <Z> DEBUGGING ONLY. Sets the position of an object. msg <ObjectName> (MOVETOPOS 50.5 100 75)
MOVETOPOS <X> <Y> <Z> DEBUGGING ONLY. Moves to the position, with collisions. msg <ObjectName> (MOVETOPOS 50.5 100 75)
SETROTATION <yaw> <pitch> <roll> DEBUGGING ONLY. Rotates object to euler angles. msg <ObjectName> (SETROTATION 90.0 0 0)
COPYXFORM <object> DEBUGGING ONLY. Copies trasform of <object> to this object. msg <ObjectName> (COPYXFORM ThePoint)
ON Tells the specified sound to be played msg SoundFX ON
OFF Tells the specified looping sound to stop playing after finishing msg SoundFX OFF

InnerRadius It's useful and simple to give a sound two radii, one Inner where it plays at full Volume, and one Outer where it plays at a reduced volume. It's a cheap way of having a sound fade over distance without requiring possibly expensive audio processing in-game. The values a set in WorldEdit units. Float (Radius) 100.000000
Volume This value is expressed as a percentage. It is how loud the sound is at its origin point. Integer 100.000000
PitchShift This value speeds up or slows down the file. You can set a different pitch with this property to make it play lower or higher in pitch. 1.00 is normal pitch, 1.10 would be pitched up 10%, 0.90 would be 10% pitched down. It does this by changing the playback sample rate of the file. Float 1.000000
Loop This flag toogles whether the sound will loop continuously or play only once when triggered on. Boolean 1.000000
Filter Every sound played in FEAR may have some type of software filtering done to it before it is passed off to the sound card. The current implementation allows sounds to independently specify what filter they use when they are played, or (by default) use the 'dynamic' filter associated with the current listener (camera) position. Also sounds can specify that they do not want to be filtered. String (StaticList) UnFiltered
MixChannel This is the mixer channel for the sound effect. Putting -1 will use the sound class's default value. Integer 1.000000
DopplerFactor This is Doppler multiplier. 1 is default (no change), and 0 turns off Doppler. Float 1.000000
