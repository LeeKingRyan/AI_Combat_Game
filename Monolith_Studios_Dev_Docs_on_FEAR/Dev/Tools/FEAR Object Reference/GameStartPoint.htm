GameStartPoint
Defines points that the player can start at in a level.

Message Description Example
VISIBLE <bool> Toggles whether the object is visible. msg <ObjectName> (VISIBLE 1)<BR>msg <ObjectName> (VISIBLE 0)
SOLID <bool> Toggles whether the object is solid. msg <ObjectName> (SOLID 1)<BR>msg <ObjectName> (SOLID 0)
HIDDEN <bool> Toggles whether the object is visible, solid, and rayhit. msg <ObjectName> (HIDDEN 1)<BR>msg <ObjectName> (HIDDEN 0)
REMOVE Removes the object from the game permanently. msg <ObjectName> REMOVE
CASTSHADOW <bool> Toggles whether the object casts a shadow. msg <ObjectName> (CASTSHADOW 1)<BR>msg <ObjectName> (CASTSHADOW 0)
SETPOS <X> <Y> <Z> DEBUGGING ONLY. Sets the position of an object. msg <ObjectName> (MOVETOPOS 50.5 100 75)
MOVETOPOS <X> <Y> <Z> DEBUGGING ONLY. Moves to the position, with collisions. msg <ObjectName> (MOVETOPOS 50.5 100 75)
SETROTATION <yaw> <pitch> <roll> DEBUGGING ONLY. Rotates object to euler angles. msg <ObjectName> (SETROTATION 90.0 0 0)
COPYXFORM <object> DEBUGGING ONLY. Copies trasform of <object> to this object. msg <ObjectName> (COPYXFORM ThePoint)
LOCK Locks the GameStarPoint so no player can enter the world using this GameStarPoint msg GameStarPoint LOCK
UNLOCK Unlocks the GameStarPoint msg GameStarPoint UNLOCK
TEAM <0, 1, -1> Specifies which team can spawn at this start point msg GameStartPoint (TEAM 1)


Property Description Type Default
Name The name of this object String noname
Pos The position in the world of this object. Vector (Distance) 0.000000, 0.000000, 0.000000
Rotation The orientation of this object represented by euler angles in degrees Rotation 0.000000, 0.000000, 0.000000
Template Indicates that this object will not exist at runtime, and may be used as an object template for Spawner objects Boolean 0.000000
IsGore Indicates that this object is gory and should not exist for low-violence clients. Boolean 0.000000
ObjectLOD This value indicates at which detail levels the object will exist. For example at Low, this object will always exist, at Medium it will only exist in medium and high detail, and Never will cause the object to never exist. String (StaticList) Low
ModelTemplate This is a dropdown menu that allows you to choose the model player model that will be loaded at the beginning of the level. String (StaticList)
PhysicsModel This dropdown menu allows you to choose what physics model is loaded at the beginning of the level. String (StaticList)
Command This is a Command string that will be processed when the player spawns in at this start point. Command
Locked This flag is used to not allow players to spawn at this start point. Sending the GameStartPoint object LOCK and UNLOCK messages will allow or disallow players from spawning at this point. Boolean 0.000000
SendCommandOnRespawn If this flag is set to true the Command string will be sent any time the player spawns at this point otherwise it will only be sent the first time. Boolean 0.000000
Team This is a dropdown that allows you to set which team can spawn at this start point while playing a team game. String (StaticList) NoTeam
StartPoint If this flag is true players will be allowed to spawn in at this start point at the very begining of the level. If this flag is false players won't be able to use this start point for their very first spawn into a level. Boolean 1.000000
SpawnPoint If this flag is true players will be allowed to respawn at this start point at any point during the game after the first spawn. If this flag is false players will not be able use this start point for respawning. Boolean 1.000000
