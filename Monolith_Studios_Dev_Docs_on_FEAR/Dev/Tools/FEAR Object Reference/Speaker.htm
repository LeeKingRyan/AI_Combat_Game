Speaker
Speaker objects are used to play dialogue from non-typical character objects, such as from a telephone or from a speaker system.

Message Description Example
VISIBLE <bool> Toggles whether the object is visible. msg <ObjectName> (VISIBLE 1)<BR>msg <ObjectName> (VISIBLE 0)
SOLID <bool> Toggles whether the object is solid. msg <ObjectName> (SOLID 1)<BR>msg <ObjectName> (SOLID 0)
HIDDEN <bool> Toggles whether the object is visible, solid, and rayhit. msg (HIDDEN 1)
msg (HIDDEN 0)
REMOVE Removes the character from the game permanently. msg REMOVE
CASTSHADOW <bool> Toggles whether the object casts a shadow. msg <ObjectName> (CASTSHADOW 1)<BR>msg <ObjectName> (CASTSHADOW 0)
SETPOS <X> <Y> <Z> DEBUGGING ONLY. Sets the position of an object. msg <ObjectName> (MOVETOPOS 50.5 100 75)
MOVETOPOS <X> <Y> <Z> DEBUGGING ONLY. Moves to the position, with collisions. msg <ObjectName> (MOVETOPOS 50.5 100 75)
SETROTATION <yaw> <pitch> <roll> DEBUGGING ONLY. Rotates object to euler angles. msg <ObjectName> (SETROTATION 90.0 0 0)
COPYXFORM <object> DEBUGGING ONLY. Copies trasform of <object> to this object. msg <ObjectName> (COPYXFORM ThePoint)
PLAYSOUND <DialogueID> Tells the character to play a line of dialogue. specifies an entry in the string database to use. msg (PLAYSOUND 15076)
TELEPORT <teleport point> | <vector> Moves the character to the position of the specified TeleportPoint object, or location. msg (TELEPORT TeleportPoint00)
msg (TELEPORT (1000, 50, 300))
ATTACH <attachment name> [attach pos] Creates an attachment on the character. specifies an Attachment record in the game database. The optional [attach pos] parameter allows you to override the default socket set in the record. msg (ATTACH Bomb)
DETACH <attachment name> [attach pos] Removes an attachment from the character. specifies an Attachment record in the game database. Use the [attach pos] parameter if the object is attached to a socket other than the default set in the record. msg (DETACH Bomb)
CANDAMAGE <bool> Sets whether the character can be damaged. msg (CANDAMAGE 0)
CROSSHAIR <alignment> Sets the characters alignment or side. The can be GOOD (friendly to the player), BAD (unfriendly), or NEUTRAL msg (CROSSHAIR BAD)
FIND Displays the position of the character in the console. Used for debugging msg FIND
MOVETOFLOOR <bool> Toggles whether the character is automatically moved to the floor. msg (MOVETOFLOOR 1)
msg (MOVETOFLOOR 0)
DETONATOR Detonates remote charges set by the character. msg Character DETONATOR
SHOW_ATTACHFX [index] Shows the attached ClientFX identified the the given index. This index refers to the array of PersistentClientFX specified in the character model's database record. msg (SHOW_ATTACHFX 1)
HIDE_ATTACHFX [index] Hides the attached ClientFX identified the the given index. This index refers to the array of PersistentClientFX specified in the character model's database record. msg (HIDE_ATTACHFX 1)
FX <ClientFX name> [socket] Plays the specified ClientFX on the character. If the optional socket is specified the ClientFX will play at that socket position. msg Character (FireFX Left_Leg);
