Explosion
Explosion objects are used to create a dynamic explosion with a ClientFX, damage type and physics impulse force.

Message Description Example
VISIBLE <bool> Toggles whether the object is visible. msg <ObjectName> (VISIBLE 1)<BR>msg <ObjectName> (VISIBLE 0)
SOLID <bool> Toggles whether the object is solid. msg <ObjectName> (SOLID 1)<BR>msg <ObjectName> (SOLID 0)
HIDDEN <bool> Toggles whether the object is visible, solid, and rayhit. msg <ObjectName> (HIDDEN 1)<BR>msg <ObjectName> (HIDDEN 0)
REMOVE Removes the object from the game permanently. msg <ObjectName> REMOVE
CASTSHADOW <bool> Toggles whether the object casts a shadow. msg <ObjectName> (CASTSHADOW 1)<BR>msg <ObjectName> (CASTSHADOW 0)
SETPOS <X> <Y> <Z> DEBUGGING ONLY. Sets the position of an object. msg <ObjectName> (MOVETOPOS 50.5 100 75)
MOVETOPOS <X> <Y> <Z> DEBUGGING ONLY. Moves to the position, with collisions. msg <ObjectName> (MOVETOPOS 50.5 100 75)
SETROTATION <yaw> <pitch> <roll> DEBUGGING ONLY. Rotates object to euler angles. msg <ObjectName> (SETROTATION 90.0 0 0)
COPYXFORM <object> DEBUGGING ONLY. Copies trasform of <object> to this object. msg <ObjectName> (COPYXFORM ThePoint)
ON Turns the explosion on. msg Explosion ON


Property Description Type Default
Name The name of this object String noname
Pos The position in the world of this object. Vector (Distance) 0.000000, 0.000000, 0.000000
Rotation The orientation of this object represented by euler angles in degrees Rotation 0.000000, 0.000000, 0.000000
Template Indicates that this object will not exist at runtime, and may be used as an object template for Spawner objects Boolean 0.000000
IsGore Indicates that this object is gory and should not exist for low-violence clients. Boolean 0.000000
ObjectLOD This value indicates at which detail levels the object will exist. For example at Low, this object will always exist, at Medium it will only exist in medium and high detail, and Never will cause the object to never exist. String (StaticList) Low
ImpactFXName Select from a list of ImpactFX listed in GameDatabase under FX\ImpactFX. String (StaticList)
DamageType This is a pull down used to identify what kind of damage will be inflicted upon anyone caught within the DamageRadius of the Explosion object when it is triggered. String (StaticList) EXPLODE
DamageRadiusMin This the minimum radius measured in WorldEdit units in which an object will take 100% of the MaxDamage damage from the explosion. The damage amount is linearly scaled from MaxDamage to 0 between the DamageRadiusMin and DamageRadius. NOTE: If set to 0, 25% of the DamageRadius will be used for the DamageRadiusMin. Float (Radius) 0.000000
DamageRadius This the radius measured in WorldEdit units in which an object will take damage from the explosion. Float (Radius) 200.000000
MaxDamage This field defines the maximum amount of damage that will be inflicted upon an object caught within the DamageRadius of the explosion when it is triggered. Float 200.000000
RemoveWhenDone This flag toggles whether or not the explosion is removed after being triggered. Sometimes you will want to trigger an Explosion object repeatedly. In which case you would set this flag to false. Boolean 1.000000
ImpulseForce The amount of force the explosion will apply to objects inside the DamageRadiusMin. The force is linearly scaled between ImpulseForce and 0 between DamageRadiusMin and DamageRadius. The value is measured in game unit newton seconds. Float 5000.000000
