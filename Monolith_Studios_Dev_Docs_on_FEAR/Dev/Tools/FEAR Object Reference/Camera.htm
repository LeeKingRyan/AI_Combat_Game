Camera
This object allows you to setup a cinematic camera.

Message Description Example
VISIBLE <bool> Toggles whether the object is visible. msg <ObjectName> (VISIBLE 1)<BR>msg <ObjectName> (VISIBLE 0)
SOLID <bool> Toggles whether the object is solid. msg <ObjectName> (SOLID 1)<BR>msg <ObjectName> (SOLID 0)
HIDDEN <bool> Toggles whether the object is visible, solid, and rayhit. msg <ObjectName> (HIDDEN 1)<BR>msg <ObjectName> (HIDDEN 0)
REMOVE Removes the object from the game permanently. msg <ObjectName> REMOVE
CASTSHADOW <bool> Toggles whether the object casts a shadow. msg <ObjectName> (CASTSHADOW 1)<BR>msg <ObjectName> (CASTSHADOW 0)
SETPOS <X> <Y> <Z> DEBUGGING ONLY. Sets the position of an object. msg <ObjectName> (MOVETOPOS 50.5 100 75)
MOVETOPOS <X> <Y> <Z> DEBUGGING ONLY. Moves to the position, with collisions. msg <ObjectName> (MOVETOPOS 50.5 100 75)
SETROTATION <yaw> <pitch> <roll> DEBUGGING ONLY. Rotates object to euler angles. msg <ObjectName> (SETROTATION 90.0 0 0)
COPYXFORM <object> DEBUGGING ONLY. Copies trasform of <object> to this object. msg <ObjectName> (COPYXFORM ThePoint)
ON Turns the camera on. Switches the view from the player to that of the camera msg Camera on
OFF Turns the camera off. Switches the view from the Camera to that of the Player msg Camera off
SKIP Turns the camera off if it can be skipped msg Camera skip
MOVETO <camerapoint> The camera will move to the specified CameraPoint object msg Camera (moveto CameraPoint1)
FOV <FovY> <FovAspectScale> [time] Changes the Camera FOV over a period of time. The time parameter is optional To change the camera FOV to an Y FOV of 60 and a normal 1.0 Aspect Scale over a period of five seconds on a Camera named "Camera", the command would look like:

msg Camera (60 1 5)

ActiveTime If you want a camera to return to the player after a set period, give the camera a time-limit property. Float -1.000000
AllowPlayerMovement Choose whether the player can move or not while the camera's running. Obviously, in most cases the camera should keep players still. Boolean 0.000000
OneTime Keeps the camera from being activated after it's been run once, even if it's triggered again. Boolean 1.000000
Type There are two types of camera: full-screen = 0 and cinematic letterboxed = 1. Integer 1.000000
StartActive Allows you to have the camera start when the level starts for an intro cutscene. Boolean 0.000000
IsListener If you want the camera to relay sounds it hears to the player instead of the sounds at the player's location, use this flag. Boolean 1.000000
CanSkip If set to true, the player can interrupt the cinematic by pressing the action key in the game. If this flag is set to false, the cinematic cannot be interrupted. Boolean 1.000000
CleanupCommand A command that will be executed when the Camera is turned off, either by a message or by the player skipping it. Command
FovY The starting Field Of View Y value. Note: You can dynamicly change the FovY and AspectScale values of a camera by sending it an FOV message. Float 78.000000
FovAspectScale The starting Field Of View Aspect Scale value. This value is used for special effects to warp the field of view in the X direction. >1 will cause the view to grow outwards, <1 will shrink the FOV in the X direction. Note: You can dynamicly change the FovY and Aspect Scale values of a camera by sending it an FOV message. Float 1.000000
OnSkipCleanupOnly If set to true, only the CleanupCommand will be sent if the player skips the cinematic. The camera will be able to be activated again even if OneTime is set to true. Boolean 0.000000
