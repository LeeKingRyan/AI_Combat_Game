WorldProperties
This object contains properties, such as fog, that help define the level.

Message Description Example
VISIBLE <bool> Toggles whether the object is visible. msg <ObjectName> (VISIBLE 1)<BR>msg <ObjectName> (VISIBLE 0)
SOLID <bool> Toggles whether the object is solid. msg <ObjectName> (SOLID 1)<BR>msg <ObjectName> (SOLID 0)
HIDDEN <bool> Toggles whether the object is visible, solid, and rayhit. msg <ObjectName> (HIDDEN 1)<BR>msg <ObjectName> (HIDDEN 0)
REMOVE Removes the object from the game permanently. msg <ObjectName> REMOVE
CASTSHADOW <bool> Toggles whether the object casts a shadow. msg <ObjectName> (CASTSHADOW 1)<BR>msg <ObjectName> (CASTSHADOW 0)
SETPOS <X> <Y> <Z> DEBUGGING ONLY. Sets the position of an object. msg <ObjectName> (MOVETOPOS 50.5 100 75)
MOVETOPOS <X> <Y> <Z> DEBUGGING ONLY. Moves to the position, with collisions. msg <ObjectName> (MOVETOPOS 50.5 100 75)
SETROTATION <yaw> <pitch> <roll> DEBUGGING ONLY. Rotates object to euler angles. msg <ObjectName> (SETROTATION 90.0 0 0)
COPYXFORM <object> DEBUGGING ONLY. Copies trasform of <object> to this object. msg <ObjectName> (COPYXFORM ThePoint)
FOGCOLOR <r> <g> <b> This message changes the color of the fog color of the world msg WorldProperties (FOGCOLOR 255 126 210)
FOGENABLE <1-or-0> This message toggles the fog in the world on and off msg WorldProperties (FOGENABLE 0)
FOGNEARZ <near z> Changes the near z of the fog plane msg WorldProperties (FOGNEARZ 300)
FOGFARZ <far z> Changes the far z of the fog plane msg WorldProperties (FOGFARZ 300)
SKYFOGENABLE <1-or-0> This message toggles the skyfog in the world on and off msg WorldProperties (SKYFOGENABLE 0)
SKYFOGFARZ <far z> Changes the far z of the skyfog plane msg WorldProperties (SKYFOGFARZ 300)
SKYFOGNEARZ <near z> Changes the near z of the skyfog plane msg WorldProperties (SKYFOGNEARZ 300)
NEXTROUND In a non-cooperative multiplayer game this message will advance the game to the next round. msg WorldProperties NEXTROUND
ROUNDWON <0 or 1> In a non-cooperative team multiplayer game this message will end the round and the team specified in the message wins the round. msg WorldProperties (ROUNDWON 1)
