GearItem
GearItem objects are used to place Game Database defined Gear records.

Message Description Example
VISIBLE <bool> Toggles whether the object is visible. msg <ObjectName> (VISIBLE 1)<BR>msg <ObjectName> (VISIBLE 0)
SOLID <bool> Toggles whether the object is solid. msg <ObjectName> (SOLID 1)<BR>msg <ObjectName> (SOLID 0)
HIDDEN <bool> Toggles whether the object is visible, solid, and rayhit. msg <ObjectName> (HIDDEN 1)<BR>msg <ObjectName> (HIDDEN 0)
REMOVE Removes the object from the game permanently. msg <ObjectName> REMOVE
CASTSHADOW <bool> Toggles whether the object casts a shadow. msg <ObjectName> (CASTSHADOW 1)<BR>msg <ObjectName> (CASTSHADOW 0)
SETPOS <X> <Y> <Z> DEBUGGING ONLY. Sets the position of an object. msg <ObjectName> (MOVETOPOS 50.5 100 75)
MOVETOPOS <X> <Y> <Z> DEBUGGING ONLY. Moves to the position, with collisions. msg <ObjectName> (MOVETOPOS 50.5 100 75)
SETROTATION <yaw> <pitch> <roll> DEBUGGING ONLY. Rotates object to euler angles. msg <ObjectName> (SETROTATION 90.0 0 0)
COPYXFORM <object> DEBUGGING ONLY. Copies trasform of <object> to this object. msg <ObjectName> (COPYXFORM ThePoint)

Property Description Type Default
Name The name of this object String noname
Pos The position in the world of this object. Vector (Distance) 0.000000, 0.000000, 0.000000
Rotation The orientation of this object represented by euler angles in degrees Rotation 0.000000, 0.000000, 0.000000
Template Indicates that this object will not exist at runtime, and may be used as an object template for Spawner objects Boolean 0.000000
IsGore Indicates that this object is gory and should not exist for low-violence clients. Boolean 0.000000
ObjectLOD This value indicates at which detail levels the object will exist. For example at Low, this object will always exist, at Medium it will only exist in medium and high detail, and Never will cause the object to never exist. String (StaticList) Low
Scale This value changes the size of the object. It is a multiplicative value based on the original size of the object. The default scale is 1.0. Float 1.000000
Visible Determines if this object is visible by default Boolean 1.000000
Shadow Determines if this object casts a shadow Boolean 1.000000
PickupCommand This is the command that is sent when the player picks up the WeaponItem. Command
SPRespawn Singleplayer: Item will respawn in RespawnTime. Boolean 0.000000
MPRespawn Multiplayer: Item will respawn in RespawnTime. Boolean 1.000000
MoveToFloor Whether or not the object is moved to the floor when created in the game. Boolean 1.000000
ModelOverride Contains a list of ModelBute objects (type=3) that can be used to override the default model. String (StaticList)
WorldAnimation Specifies the name of the animation to use for the model. The default 'World' animation is typically lying flat on the ground. The 'BinPos' animation should be an animation created in model edit with the 'Create anim from Bind Pose' menu option. The model that is visible in WorldEdit is using the models bind pose so setting this property to 'BindPos' will allow LD's to rotate the model and have it look exactly the same in game as it does in WorldEdit. String (DynamicList) World
Team This is a dropdown list that allows you to specify if a team owns this item. Only players on the owning team will be able to pickup this item. If NoTeam is selected then all players can pickup the item. String (StaticList) NoTeam
Locked Determines if this pickup starts out as locked (i.e. can't be picked up). Boolean 0.000000
GearType A pull down menu used to choose the type of GearItem the power up will be. String (Dims LocalDims StaticList)
GlobalResources Indicates whether the object's resources should be considered as global in the resource streaming system. Boolean 0.000000
