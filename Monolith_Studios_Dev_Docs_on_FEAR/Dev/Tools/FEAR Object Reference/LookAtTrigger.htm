LookAtTrigger
Binding brushes to this object creates a space that will execute commands when looked at.

Message Description Example
VISIBLE <bool> Toggles whether the object is visible. msg <ObjectName> (VISIBLE 1)<BR>msg <ObjectName> (VISIBLE 0)
SOLID <bool> Toggles whether the object is solid. msg <ObjectName> (SOLID 1)<BR>msg <ObjectName> (SOLID 0)
HIDDEN <bool> Toggles whether the object is visible, solid, and rayhit. msg <ObjectName> (HIDDEN 1)<BR>msg <ObjectName> (HIDDEN 0)
REMOVE Removes the object from the game permanently. msg <ObjectName> REMOVE
CASTSHADOW <bool> Toggles whether the object casts a shadow. msg <ObjectName> (CASTSHADOW 1)<BR>msg <ObjectName> (CASTSHADOW 0)
SETPOS <X> <Y> <Z> DEBUGGING ONLY. Sets the position of an object. msg <ObjectName> (MOVETOPOS 50.5 100 75)
MOVETOPOS <X> <Y> <Z> DEBUGGING ONLY. Moves to the position, with collisions. msg <ObjectName> (MOVETOPOS 50.5 100 75)
SETROTATION <yaw> <pitch> <roll> DEBUGGING ONLY. Rotates object to euler angles. msg <ObjectName> (SETROTATION 90.0 0 0)
COPYXFORM <object> DEBUGGING ONLY. Copies trasform of <object> to this object. msg <ObjectName> (COPYXFORM ThePoint)
LOCK <..., LookAt, LookAway> Locks the entire Trigger or individual components so that it ignores all commands other than UNLOCK and REMOVE and cannot be activated by the player msg Trigger LOCK;
msg Trigger (LOCK LookAt);
msg Trigger (LOCK LookAway);
UNLOCK <...,, LookAt, LookAway> Unlocks the entire trigger or individual components so that the player or other messages can activate it again msg Trigger UNLOCK;
msg Trigger (UNLOCK LookAt);
msg Trigger (UNLOCK LookAway);
LOOKAT Sets the object to it�s LookAt state, ignoring threshold, executing the LookAt command and incrementing the number of lookats msg Trigger LOOKAT;
LOOKAWAY Sets the object to it�s LookAway state, ignoring threshold, executing the LookAway msg Trigger LOOKAWAY;

LookAwayThreshold Amount of time, in seconds, a player must be looking away from the object before they are considered looking away from it. Set to 0.0 for immediate execution of LookAway command. Float 0.000000
LookAwayCommand This command will be executed when a player is considered looking away from the object. Command
